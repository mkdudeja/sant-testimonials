## Project Setup

- `npm install` - To install the packages
- `npm start` - To run the project

## Backend Integration

### Set the BE API_URL

- Open chrome devtools using F12
- Goto `Application` tab
- Goto `Storage` => `localStorage` tab (must be on the left side)
- Select our application URL (ex: http://localhost:3000)
- On the right hand side, create variable
  - key => `apiUrl`
  - value => `{BACKEND_API_URL}`
